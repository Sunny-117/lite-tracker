## 指令

- pwd 打印当前目录
- ifconfig 查看 ip 地址和网络名

setup 图形化配置

- 认证配置
- 防火墙配置
- 网络配置
- 系统服务

网络

- ifup eth0 网卡接口激活
- ifdown eth0 网卡接口关闭
- service network restart 重启网络服务

## 权限

r 读取 4
w 写入 2
x 执行 1
`-` 无 0

## 硬链接和软链接

硬链接 `In [源文件] [目标文件]`
软链接 `In -s [源文件] [目标文件]`

## 格式化

- 把分区切分成等大小的数据块(多个物理扇区)，每个数据块是 4k，10k 需要使用 2 个半的数据块。是存放文件的最小空间。（dos、windows 中叫簇）
- 划分文件分配表，目录表。id、修改时间、权限、数据块位置
